\# Banking Microservices System



Proyecto backend basado en microservicios desarrollado con Java, Spring Boot, MySQL y Spring Cloud.



\## Objetivo



Construir una arquitectura de microservicios orientada a un sistema bancario, separando responsabilidades por dominio como clientes, cuentas, autenticación e infraestructura.
